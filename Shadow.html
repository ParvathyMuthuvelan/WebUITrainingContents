The box-shadow Property

The box-shadow property defines the shadow of an element:

box-shadow: 15px 10px:

This is a div element with a shadow. First value is the horizontal offset and the second value is the vertical offset. The shadow color will be inherited from the text color.

box-shadow: 15px 15px 10px #4286f4:

You can also define the color for the shadow. Here the shadow color is blue.

box-shadow: 15px 15px 10px red:

A red shadow.
